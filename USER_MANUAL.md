## 1. 앱 소개

**T-CARDLY**는 AI 기반의 스마트 명함 관리 앱입니다. 카메라로 명함을 촬영하면 OCR(광학 문자 인식) 기술이 한국어 텍스트를 자동으로 인식하여 디지털화합니다. 단순한 명함 저장을 넘어 기업 정보 조회, AI 기업 분석, SWOT 분석 등 비즈니스 인사이트를 제공합니다.

### 주요 특징

- **스마트 명함 스캔**: 카메라, QR 코드, 갤러리 이미지를 통한 명함 인식
- **한국어 OCR 최적화**: ML Kit 기반 한국어 텍스트 정밀 인식
- **기업 정보 자동 조회**: 대표자, 업종, 재무 정보 등 기업 상세 정보 확인
- **AI 기업 분석**: SWOT 분석, 투자 매력도 점수, 사업 전망 등 AI 기반 분석
- **AI 채팅**: 기업에 대한 궁금한 점을 AI에게 자유롭게 질문
- **그룹·태그 분류**: 체계적인 명함 분류 및 관리
- **데이터 내보내기**: CSV, vCard 형식으로 명함 데이터 내보내기
- **암호화 저장**: SQLCipher를 통한 로컬 데이터 암호화 보호
- **오프라인 지원**: 인터넷 연결 없이도 기본 기능 사용 가능

## 2. 시작하기

### 2.1 앱 설치 및 실행

1. Google Play 스토어에서 **T-CARDLY**를 검색합니다.
2. **설치** 버튼을 눌러 앱을 다운로드합니다.
3. 설치 완료 후 **열기** 버튼을 눌러 앱을 실행합니다.

### 2.2 초기 프로필 설정

앱을 처음 실행하면 **프로필 설정** 화면이 표시됩니다.

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 이름 | **필수** | 본인의 이름을 입력합니다 |
| 현재 회사 | 선택 | 소속 회사명을 입력합니다 |
| 직책 | 선택 | 현재 직책을 입력합니다 |
| 휴대전화 | 선택 | 휴대전화 번호를 입력합니다 |
| 이메일 | 선택 | 이메일 주소를 입력합니다 |

- **시작하기** 버튼을 눌러 프로필을 저장하고 홈 화면으로 이동합니다.
- 프로필 설정을 건너뛰려면 **나중에 설정할게요**를 눌러주세요. 이후 전체 메뉴 > 프로필에서 언제든 설정할 수 있습니다.

### 2.3 필요 권한 안내

앱 사용 시 다음 권한이 필요합니다. 각 기능을 처음 사용할 때 권한 요청 팝업이 나타납니다.

| 권한 | 용도 |
|------|------|
| 카메라 | 명함 촬영 및 QR 코드 스캔 |
| 사진/미디어 접근 | 갤러리에서 명함 이미지 불러오기 |
| 인터넷 | AI 분석, 기업 정보 조회 |
| 연락처 접근 | 연락처 정보 연동 |

## 3. 홈 화면

앱 실행 시 가장 먼저 보이는 대시보드 화면입니다.

### 화면 구성

- **상단 영역**: T-CARDLY 로고와 프로필 아이콘
- **인사 메시지**: 시간대에 따른 인사말과 보유 명함 수 표시
- **최근 등록 명함**: 최근에 추가한 명함 목록
- **Pro 업그레이드 배너**: 무료 플랜 사용자에게 표시되는 Pro 혜택 안내 (광고 제거, AI 분석 30회/월, 무제한 내보내기 등)

### 주요 동작

| 동작 | 설명 |
|------|------|
| 프로필 아이콘 탭 | 내 프로필 화면으로 이동 |
| 카메라 버튼 (FAB) 탭 | 명함 스캔 화면으로 이동 |
| 명함 카드 탭 | 해당 명함 상세 화면으로 이동 |
| 회사명 탭 | 기업 상세 정보 화면으로 이동 |
| 업그레이드 버튼 탭 | 구독 요금제 화면으로 이동 |

명함이 없는 경우 **"아직 등록된 명함이 없어요"** 메시지와 함께 **첫 명함 스캔하기** 버튼이 표시됩니다.

## 4. 명함 스캔

하단 내비게이션의 **스캔** 탭 또는 홈 화면의 카메라 버튼을 눌러 명함 스캔 화면에 진입합니다.

### 4.1 카메라 촬영

1. 스캔 화면에서 **촬영** 모드가 기본 선택됩니다.
2. 명함을 화면에 표시된 가이드 프레임 안에 맞춥니다.
   - 화면에 **"명함을 프레임 안에 맞춰주세요"** 안내 문구가 표시됩니다.
3. 하단의 **촬영 버튼**(원형 틸 버튼)을 눌러 촬영합니다.
4. 어두운 환경에서는 **플래시 아이콘**을 눌러 플래시를 켤 수 있습니다.
5. 촬영 후 OCR 인식이 진행되며 **"명함을 인식하고 있습니다..."** 메시지가 표시됩니다.
6. 인식이 완료되면 자동으로 **스캔 결과 확인** 화면으로 이동합니다.

### 4.2 QR 코드 스캔

1. 하단 모드 탭에서 **QR**을 선택합니다.
2. **"QR 코드를 스캔해 주세요"** 안내 문구가 표시됩니다.
3. QR 코드를 가이드 프레임 안에 맞추면 자동으로 인식됩니다.
4. vCard 형식의 QR 코드에서 이름, 회사, 직책, 연락처, 이메일, 주소, 웹사이트 정보를 자동 추출합니다.
5. 인식이 완료되면 **스캔 결과 확인** 화면으로 이동합니다.

### 4.3 갤러리에서 불러오기

1. 하단 모드 탭에서 **갤러리**를 선택합니다.
2. **"갤러리에서 이미지를 선택해 주세요"** 안내 문구가 표시됩니다.
3. 하단의 **갤러리 버튼**을 누르면 사진 선택 화면이 열립니다.
4. 명함 이미지를 선택하면 OCR 인식이 자동으로 시작됩니다.
5. 인식이 완료되면 **스캔 결과 확인** 화면으로 이동합니다.

### 4.4 스캔 결과 확인 및 편집

OCR 인식 완료 후 표시되는 화면입니다.

#### 화면 구성

- **인식 정확도**: 상단에 OCR 인식 신뢰도가 배지로 표시됩니다.
- **편집 가능 필드**: 아래 10개 항목을 확인하고 수정할 수 있습니다.

| 필드 | 아이콘 | 설명 |
|------|--------|------|
| 이름 (*필수) | 사람 | 명함 소유자의 이름 |
| 회사명 | 건물 | 소속 회사 |
| 직책 | 뱃지 | 직책/직위 |
| 부서 | 그룹 | 소속 부서 |
| 휴대전화 | 전화 | 휴대전화 번호 |
| 사무실 전화 | 전화 | 사무실 전화 번호 |
| 이메일 | 메일 | 이메일 주소 |
| 주소 | 위치 | 사무실/회사 주소 |
| 웹사이트 | 지구 | 회사 웹사이트 |
| 메모 | 노트 | 자유 메모 |

#### 저장 방법

1. OCR이 자동으로 채운 각 항목을 확인합니다.
2. 잘못 인식된 부분이 있으면 해당 필드를 탭하여 수정합니다.
3. **이름** 항목은 필수이며, 비어있으면 저장할 수 없습니다.
4. **명함 저장** 버튼을 눌러 저장합니다.
5. 저장이 완료되면 자동으로 홈 화면으로 이동합니다.

### 4.5 수기 입력

OCR 스캔 없이 명함 정보를 직접 입력할 수 있습니다.

1. 스캔 화면 상단의 **편집 아이콘**을 탭합니다.
2. 또는 연락처 화면에서 **+** 버튼 > **수기 입력**을 선택합니다.
3. **명함 수기 입력** 화면이 열립니다.

#### 입력 섹션

**기본 정보**
- 이름 (*필수), 회사명, 직책, 부서

**연락처**
- 휴대전화, 사무실 전화, 이메일, 주소, 웹사이트

**메모**
- 자유롭게 메모를 작성할 수 있습니다.

4. 모든 정보를 입력한 후 **명함 저장** 버튼을 누릅니다.

## 5. 연락처 관리

하단 내비게이션의 **연락처** 탭에서 저장된 모든 명함을 관리합니다.

### 5.1 연락처 목록

저장된 명함이 목록 형태로 표시됩니다. 각 항목에는 다음 정보가 나타납니다:
- 프로필 아바타 (이니셜 표시)
- 이름
- 회사명
- 직책

명함이 없는 경우 **"등록된 명함이 없습니다"** 메시지가 표시됩니다.

#### 명함 추가

1. 화면 우측 하단의 **+** 버튼(FAB)을 탭합니다.
2. 하단 시트에서 추가 방법을 선택합니다:
   - **스캔으로 추가**: 스캔 화면으로 이동
   - **수기 입력**: 수기 입력 화면으로 이동

### 5.2 검색 및 필터링

#### 검색

- 화면 상단의 검색창에 텍스트를 입력합니다.
- **이름, 회사, 전화번호**로 검색할 수 있습니다.
- 입력 즉시 실시간으로 결과가 필터링됩니다.

#### 태그 필터

- 검색창 아래에 태그 필터 칩이 가로로 스크롤 가능하게 표시됩니다.
- 원하는 태그를 탭하면 해당 태그가 지정된 명함만 필터링됩니다.
- 탭한 태그를 다시 탭하면 필터가 해제됩니다.
- 여러 태그를 동시에 선택하여 복합 필터링이 가능합니다.

### 5.3 명함 상세 보기

연락처 목록에서 명함을 탭하면 상세 화면으로 이동합니다.

#### 프로필 헤더

- 원형 아바타 (이름 이니셜 표시)
- 이름 (큰 글씨)
- 직책 · 부서
- 회사명 (탭하면 기업 상세 정보로 이동)
- 태그 칩 (지정된 태그 표시)
- OCR 인식 정확도 배지

#### 연락처 정보

| 항목 | 아이콘 |
|------|--------|
| 휴대전화 | 전화 |
| 사무실 전화 | 전화 |
| 이메일 | 메일 |
| 주소 | 위치 |
| 웹사이트 | 지구 |

#### 메모

저장된 메모가 있는 경우 메모 섹션이 표시됩니다.

#### AI 기업 분석 버튼

회사명이 등록된 명함의 경우 하단에 **AI 기업 분석** 버튼이 표시됩니다. 탭하면 AI 분석 화면으로 이동합니다.

### 5.4 즐겨찾기

명함 상세 화면 상단의 **별 아이콘**을 탭하면 즐겨찾기를 설정/해제할 수 있습니다.
- 즐겨찾기 설정: 별 아이콘이 채워지고 코랄색으로 변경
- 즐겨찾기 해제: 별 아이콘이 비어있고 회색으로 변경

### 5.5 명함 공유

명함 상세 화면 상단의 **공유 아이콘**을 탭하면 명함 정보를 다른 앱으로 공유할 수 있습니다.

## 6. 기업 정보 및 AI 분석

### 6.1 기업 상세 정보

명함의 회사명을 탭하면 해당 기업의 상세 정보를 확인할 수 있습니다.

#### 기본 정보

| 항목 | 설명 |
|------|------|
| 대표 | 대표이사 이름 |
| 업종 | 사업 분야 |
| 설립일 | 회사 설립 날짜 |
| 직원 수 | 전체 직원 수 |
| 주소 | 본사 주소 |
| 웹사이트 | 회사 공식 웹사이트 |
| 신용등급 | 기업 신용 등급 |
| 상장 여부 | 상장/비상장 구분 |

#### 재무 정보

연도별 재무 데이터가 최신순으로 표시됩니다:
- **매출**: 연간 총매출액
- **영업이익**: 연간 영업이익
- **순이익**: 연간 순이익

금액은 한국 표준 단위(조, 억원, 만원, 원)로 표시됩니다.

#### 하단 액션 버튼

- **AI 기업 분석**: AI 기반 기업 종합 분석 화면으로 이동
- **AI에게 질문하기**: 해당 기업에 대해 AI와 대화하는 채팅 화면으로 이동

### 6.2 AI 기업 분석

기업에 대한 AI 기반 종합 분석 리포트를 제공합니다.

#### 투자 매력도 점수

- 0~100점 척도로 기업의 투자 매력도를 평가합니다.
- 색상 코드:
  - **초록색** (70점 이상): 높은 투자 매력도
  - **노란색** (40~69점): 보통 투자 매력도
  - **빨간색** (40점 미만): 낮은 투자 매력도

#### AI 요약

AI가 기업의 전반적인 상황을 요약하여 제공합니다.

#### SWOT 분석

| 구분 | 색상 | 설명 |
|------|------|------|
| 강점 (Strengths) | 초록 | 기업의 핵심 경쟁력 |
| 약점 (Weaknesses) | 빨강 | 개선이 필요한 부분 |
| 기회 (Opportunities) | 파랑 | 활용 가능한 외부 요인 |
| 위협 (Threats) | 노랑 | 주의해야 할 외부 위험 요소 |

각 항목은 항목별로 구분되어 리스트 형태로 제공됩니다.

#### 사업 전망

기업의 미래 사업 전망에 대한 AI 분석 내용이 표시됩니다.

#### 리스크 이슈

기업이 직면할 수 있는 잠재적 리스크와 이슈가 표시됩니다.

#### 사용 한도

- **무료 플랜**: AI 분석 기능 제한적
- **Pro 플랜**: 월 30회 AI 분석 가능
- **Business 플랜**: 무제한 AI 분석

한도 초과 시 **"AI 분석 한도 초과"** 메시지와 함께 **Pro 플랜 보기** 업그레이드 버튼이 표시됩니다.

#### 추가 질문

분석 결과 하단의 **AI에게 추가 질문하기** 버튼을 눌러 AI 채팅으로 이동할 수 있습니다.

### 6.3 AI 채팅

특정 기업에 대해 AI와 자유롭게 대화할 수 있는 채팅 기능입니다.

#### 사용 방법

1. 기업 상세 정보 또는 AI 분석 화면에서 **AI에게 질문하기** 버튼을 탭합니다.
2. 채팅 화면이 열리며 **"{회사명}에 대해 궁금한 점을 자유롭게 질문해 보세요!"** 안내 메시지가 표시됩니다.
3. 하단 입력창에 질문을 입력합니다.
4. **전송 버튼**을 탭하면 질문이 전송됩니다.
5. **"AI가 답변을 준비하고 있습니다..."** 메시지가 표시된 후 AI의 답변이 나타납니다.

#### 채팅 인터페이스

- **내 질문**: 우측 정렬, 틸(teal) 배경, 흰색 텍스트
- **AI 답변**: 좌측 정렬, 흰색 배경, 진한 텍스트
- 대화 내역은 스크롤하여 확인할 수 있으며, 새 메시지가 오면 자동으로 최신 메시지로 스크롤됩니다.

## 7. 전체 메뉴

하단 내비게이션의 **전체** 탭에서 앱의 전체 설정과 관리 기능에 접근합니다.

### 메뉴 화면 구성

- **프로필 카드**: 이름, 구독 플랜 배지, 회사명 표시. 탭하면 프로필 화면으로 이동
- **통계**: 보유 명함 총 수
- **Pro 업그레이드 배너** (무료 플랜만 표시): 광고 제거 · AI 분석 30회 · 무제한 내보내기 혜택 안내
- **관리 메뉴**:
  - 구독 관리
  - 그룹·태그 관리
- **설정 메뉴**:
  - 앱 설정

### 7.1 그룹·태그 관리

명함을 체계적으로 분류하기 위한 그룹과 태그를 관리합니다.

#### 그룹 관리

그룹은 명함을 큰 범주로 분류하는 데 사용됩니다.

- **그룹 추가**: 그룹 섹션의 **+** 버튼을 탭 > 그룹 이름 입력 > **추가** 버튼
- **그룹 삭제**: 그룹 항목의 **삭제** 아이콘을 탭 (기본 그룹은 삭제 불가)

#### 태그 관리

태그는 명함에 세부 분류 표시를 추가하는 데 사용됩니다.

- **태그 추가**: 태그 섹션의 **+** 버튼을 탭 > 태그 이름 입력 > **추가** 버튼
- **태그 삭제**: 태그 항목의 **삭제** 아이콘을 탭 (기본 태그는 삭제 불가)

태그는 각각 고유한 배경색과 텍스트 색상을 가지며, 명함 상세 화면에서 칩 형태로 표시됩니다.

### 7.2 앱 설정

전체 메뉴 > **앱 설정**에서 다양한 앱 환경을 설정할 수 있습니다.

#### 일반 설정

| 설정 | 설명 |
|------|------|
| 다크 모드 | 어두운 테마로 전환합니다 |
| 알림 | 푸시 알림을 켜거나 끕니다 |

#### 데이터 관리

| 설정 | 설명 |
|------|------|
| 데이터 백업 | 명함 데이터를 백업합니다 |
| 데이터 복원 | 백업한 데이터를 복원합니다 |
| 전체 명함 내보내기 (CSV) | 모든 명함을 CSV 파일로 내보냅니다 |
| 전체 명함 내보내기 (vCard) | 모든 명함을 vCard 파일로 내보냅니다 |

#### 정보

| 항목 | 설명 |
|------|------|
| 앱 버전 | 현재 앱 버전 정보 |
| 이용약관 | 서비스 이용약관 확인 |
| 개인정보 처리방침 | 개인정보 처리 정책 확인 |
| 오픈소스 라이선스 | 사용된 오픈소스 라이선스 정보 확인 |

## 8. 내 프로필

전체 메뉴 또는 홈 화면의 프로필 아이콘을 탭하여 프로필 화면에 접근합니다.

### 프로필 보기

- **프로필 헤더**: 원형 아바타, 이름, 직책 · 회사명
- **연락처 정보**: 휴대전화, 이메일
- **경력 사항**: 회사명, 직책, 근무 기간 (현재 재직 중인 경우 "현재"로 표시)

### 프로필 수정

1. 화면 상단 우측의 **편집 아이콘**(연필 모양)을 탭합니다.
2. 편집 모드로 전환되며 아래 항목을 수정할 수 있습니다:
   - 이름 (*필수)
   - 현재 회사
   - 현재 직책
   - 휴대전화
   - 이메일
3. 수정 완료 후 상단 우측의 **체크 아이콘**을 탭하여 저장합니다.

  - AI 기업 분석: 사용 횟수 / 한도

- Business 사용자: 업그레이드 버튼 없음 (최상위 플랜)

## 10. 데이터 내보내기

앱 설정 > 데이터 관리에서 명함 데이터를 외부 파일로 내보낼 수 있습니다.

### CSV 내보내기

엑셀에서 열 수 있는 CSV 형식으로 모든 명함을 내보냅니다. 명함의 이름, 회사, 직책, 연락처 등 모든 필드가 포함됩니다.

### vCard 내보내기

표준 vCard(.vcf) 형식으로 내보냅니다. 내보낸 파일은 다른 연락처 앱(예: Google 연락처, Apple 연락처)에서 바로 가져올 수 있습니다.

## 11. 보안 및 개인정보

T-CARDLY는 사용자의 데이터 보안을 최우선으로 합니다.

### 데이터 암호화

- 모든 명함 데이터는 **SQLCipher**를 사용하여 로컬 데이터베이스에 암호화 저장됩니다.
- 민감한 설정 정보는 **AndroidX Security Crypto**를 통해 암호화된 SharedPreferences에 저장됩니다.

### 로컬 우선 아키텍처

- 모든 명함 데이터는 기기 내에 로컬로 저장됩니다.
- 인터넷 연결이 필요한 기능은 AI 분석, 기업 정보 조회 등 외부 데이터 요청 시에만 사용됩니다.
- 오프라인 상태에서도 명함 저장, 검색, 열람 등 핵심 기능을 사용할 수 있습니다.

### 권한 최소화

앱은 필요한 최소한의 권한만 요청하며, 각 권한은 해당 기능을 처음 사용할 때만 요청됩니다.

## 12. 문제 해결

### 자주 묻는 질문

**Q: 명함 스캔 시 글자가 제대로 인식되지 않아요.**
> - 밝은 조건에서 촬영해 주세요. 플래시를 켜면 도움이 됩니다.
> - 명함을 가이드 프레임 안에 평평하게 맞춰주세요.
> - 촬영 후 스캔 결과 화면에서 잘못 인식된 부분을 직접 수정할 수 있습니다.
> - 갤러리에서 깨끗한 이미지를 선택하여 다시 시도해 보세요.

**Q: AI 기업 분석이 되지 않아요.**
> - 인터넷 연결 상태를 확인해 주세요.
> - 무료 또는 Pro 플랜의 월간 분석 한도를 초과했을 수 있습니다. 구독 관리에서 사용량을 확인해 주세요.
> - 오류가 발생한 경우 **다시 시도** 버튼을 눌러주세요.

**Q: 기존 구독이 표시되지 않아요.**
> - 전체 메뉴 > 구독 관리 또는 구독 화면에서 **구매 복원** 버튼을 탭해 주세요.
> - 구독 시 사용한 동일한 Google 계정으로 로그인되어 있는지 확인해 주세요.

**Q: 명함 데이터를 다른 기기로 옮기고 싶어요.**
> - 앱 설정 > 데이터 백업으로 현재 데이터를 백업한 후, 새 기기에서 데이터 복원을 사용해 주세요.
> - 또는 CSV/vCard로 내보내기하여 새 기기에서 가져올 수 있습니다.

**Q: 그룹이나 태그를 삭제할 수 없어요.**
> - 기본 그룹과 기본 태그는 삭제할 수 없습니다. 사용자가 직접 추가한 그룹과 태그만 삭제 가능합니다.
